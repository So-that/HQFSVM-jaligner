# HQFSVM-SA

### 环境
***********
* jdk 1.8
* libsvm
* weka >=3.8

### 使用方法
***********************
* 输入文件：arff格式
* 命令行：java &emsp-jar &emsp HQFSVM-SA.jar  &emsp -f  &emsp trainfile &emsp  -p &emsp testfile &emsp -c &emsp cv
* 测试用例：见example文件夹
